# scripts/ — SPI Flash 烧录工具

> 本目录仅保留两种烧录工具：全量烧录 LFS 文件系统和全片擦除 SPI Flash。

---

## 全量烧录 LFS 到 SPI Flash

### 场景

首次组装电路板、更换 Flash 芯片、恢复出厂状态时使用。将整个 LittleFS 文件系统镜像（含 6 张图片）写入 W25Q128。

### 操作步骤

```bash
# 第 1 步：从图片源文件生成 LFS 镜像（仅首次或图片变更时需要）
python 工具-从图片生成LFS镜像.py
# 读取 Core/Src/images/ui_img_*.c → mklfs_host 创建 → lfs_img.bin (14MB)

# 第 2 步：一键全量烧录
# 自动完成：生成 batch 头 → 擦除 Flash → 编译 固件-全量烧录.c → 分批烧录
0-全量烧录LFS到SPIFlash.bat

# 验证：上电后串口应看到
#   [IMG] Loading images from LFS to SRAM...
#   [IMG] Loaded 45292 bytes to SRAM (6/6 images)
```

### 内部过程

```
0-全量烧录LFS到SPIFlash.bat
  ├─ 工具-拆分LFS镜像为批次头文件.py
  │   lfs_img.bin → 28 × 512KB (batch_00.h ~ batch_27.h)
  │
  ├─ 烧录 工具-SPIFlash全片擦除.hex → MCU → 全片擦除 (~80s)
  │
  └─ 循环 28 次:
      编译 固件-全量烧录.c + batch_XX.h → batch_XX.hex
      STM32_Programmer_CLI 烧录 → MCU 复位 → SPI1 写入 512KB
```

### 耗时参考

| 阶段 | 耗时 |
|---|---|
| 生成 lfs_img.bin | ~2s |
| 全片擦除 (CMD_CE) | ~80s |
| 28 批次编译 | ~30s |
| 28 批次烧录 (CubeProgrammer) | ~4min |
| **合计** | **~6min** |

## 全片擦除 SPI Flash

### 场景

Flash 内容混乱需要彻底清空、更换 Flash 芯片、回滚出厂状态。

### 操作

```bash
# 烧录擦除固件 → MCU 运行 → 全片擦除 (~80s) → 串口输出 ERASE DONE
2-工具-全片擦除SPIFlash.bat
```

## 文件清单

### 可执行入口

| 文件 | 功能 | 调用方式 |
|---|---|---|
| `0-全量烧录LFS到SPIFlash.bat` | 全量初始化 LFS | 双击运行 |
| `2-工具-全片擦除SPIFlash.bat` | 擦除整个 SPI Flash | 双击运行 |
| `工具-从图片生成LFS镜像.py` | 从 ui_img_*.c 生成 lfs_img.bin | `python ...py` |
| `工具-拆分LFS镜像为批次头文件.py` | 将 lfs_img.bin 拆分为批次 | `python ...py`（自动） |

### 固件源码

| 文件 | 用途 |
|---|---|
| `固件-全量烧录.c` | 写入整块数据到 SPI Flash（编译中间件） |
| `工具-SPIFlash全片擦除.c/.elf/.hex` | 全片擦除固件 |

### PC 端工具

| 文件 | 用途 |
|---|---|
| `mklfs_host.c` | LFS 镜像创建工具源码（由 工具-从图片生成LFS镜像.py 自动编译） |
| `stm32f407_batch.ld` | 独立裸机固件链接脚本 |
| `lfs_img.bin` | LittleFS 文件系统镜像 (14MB) |
| `batch_00.h ~ batch_27.h` | 28 批次数据头文件 (512KB × 28) |

## 数据流

```
Core/Src/images/ui_img_*.c
  ↓
工具-从图片生成LFS镜像.py
  ↓
lfs_img.bin (14MB LittleFS 镜像)
  ↓
工具-拆分LFS镜像为批次头文件.py
  ↓
batch_00.h ~ batch_27.h (28 × 512KB)
  ↓
0-全量烧录LFS到SPIFlash.bat
  ├─ 擦除 SPI Flash (工具-SPIFlash全片擦除.hex)
  └─ 循环 28 次:
      编译 固件-全量烧录.c + batch_XX.h
      → STM32_Programmer_CLI → MCU → SPI1 → W25Q128
  ↓
W25Q128: LittleFS (gauge.bin, needle.bin, light.bin, ...)
  ↓
APP 启动 → spi_img_load_all() → LFS → 外部 SRAM → LVGL 渲染
```
